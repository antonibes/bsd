## Overview
A high-performance financial analytics platform designed for real-time cashflow monitoring and predictive expenditure forecasting.

**Author:** [Antoni Beśka](https://github.com/antonibes)

## Core Capabilities
*   **Dynamic Data Ingestion**: Seamless parsing of bank exports (CSV/XLSX).
*   **Intelligent Categorization**: Proprietary regex-based engine.
*   **Predictive Analytics**: 30-day liquidity forecasting.
*   **Interactive Visualizations**: Real-time Plotly charts.
*   **Containerized Architecture**: Production-ready Docker configuration.

## Getting Started
1. Clone & cd
2. `python3 -m pip install -r requirements.txt`
3. `python3 -m streamlit run app.py`
